# SQL Analytics example (Python + SQL)

This project demonstrates practical SQL techniques for data analysis using a small e-commerce style database.  
All analytical logic is implemented in SQL queries, while Python is used to execute them and display results.


## Tech Stack

- Python
- SQLite
- SQL

---

## Queries
1. **Joins** — Full order details across all tables
2. **Aggregations** — Customers with more than one order
3. **Subqueries** — Products priced above category average
4. **Window Functions** — Top 2 products per category
5. **LAG** — Monthly revenue with month-over-month change
6. **ROW_NUMBER** — First order per customer
7. **CTE** — Customer segmentation funnel
8. **Best Customer** — Top spender per category
